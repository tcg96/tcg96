Computers

Lenovo ThinkPad T440s laptop from 2014

  • 2.7 GHz Intel Core i7-4600U (dual-core with hyper-threading)
  • 12 GB 1600 MHz DDR3 RAM
  • 256 GB Samsung SSD
  • Intel HD Graphics 4400
  • 14-inch 1920*1080 IPS LCD
  • Running Windows 10 Pro 64-bit & Mac OS 10.14 Mojave

This computer has almost the same specifications as my HP 15 laptop (shown below), but it has a much better screen and some other extra features.
I only use Mac OS on this computer for video editing (with iMovie). For everything else I use Windows.

HP 15 laptop from 2015

  • 2.6 GHz Intel Core i7-4510U (dual-core with hyper-threading)
  • 4 GB 1600 MHz DDR3 RAM
  • 500 GB Samsung SSD 850 EVO
  • Intel HD Graphics 4400 & NVIDIA GeForce 820M with 2 GB VRAM
  • 15-inch 1366*768 LCD
  • Running Windows 10 Pro 64-bit & Ubuntu 19.04 Disco Dingo 64-bit

This was my main computer from May 2015 to February 2019, when I got the ThinkPad T440s. Both computers had 8 GB of RAM at first, but I moved RAM sticks around, upgrading the T440s to 12 GB of RAM (4+8) and downgrading the HP 15 to 4 GB.
With half the memory it used to have, this computer is a bit slower, but it's usable and can still run some programs (like video games) better, thanks to the NVIDIA dedicated graphics.

Apple MacBook Pro (15-inch, 2.53 GHz, mid 2009)

  • 2.53 GHz Intel Core 2 Duo P8700
  • 8 GB 1067 MHz DDR3 RAM
  • 240 GB SanDisk SSD Plus
  • NVIDIA GeForce 9400M with 256 MB VRAM
  • 15-inch 1440*900 LCD
  • Running Mac OS X 10.11 El Capitan

I've had this computer run Mac OS 10.12 Sierra and 10.13 High Sierra, which worked fine, but in September 2018 I went back to El Capitan because I prefer the latest version officially supported on this Mac.
This computer is mostly equivalent in specs to a ThinkPad T400/T500.

HP Stream 13 laptop from 2016 (family PC)

  • 32 GB eMMC
  • Intel HD Graphics
  • 13-inch 1366*768 LCD

The new Windows 10 version 1903 (May 2019 update) reserves some disk space for low-space devices like this, so that all future updates should be able to be installed with no trouble. We'll see if that's the case, because I'm not keeping an operating system that can't be updated.

HP OmniBook 2000CS laptop from 1997

  • 133 MHz Intel Pentium
  • 16 MB RAM
  • 4 GB SD card in adapter (originally: 1 GB HDD)
  • Chips and Technologies 65554 with 2 MB VRAM
  • 12-inch 800*600 passive-matrix LCD
  • Running Windows 95 OSR 2

This computer has the power board and keyboard controller integrated into one circuit board, which appears to be a point of failure because this computer often fails to turn on or the keyboard is unresponsive. For this reason I rarely use it.

My p5.js stuff

p5.js is Processing (programming language for visual art) for your web browser using JavaScript instead of Java.
Keep in mind Java and JavaScript are different and not related, they just look similar because it's basic calls to the same functions provided by Processing/p5.js. Also I'm not an artist.

    The new Patreon logo is so basic, I knew I could replicate it in Processing in less than 5 minutes. Eventually, I spent an hour trying to make it look as accurate as possible and adding the versions with other colors (as shown on their guidelines page), but it's not perfect.

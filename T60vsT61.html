ThinkPad T60 vs. T61 Windows XP recovery comparison

In July 2019 I made a video showing the restoration of Windows XP on the ThinkPad T60. However, if you want to install Windows XP on your compatible ThinkPad, you might want to use the T61 version instead of the T60 version.
They are very similar: both Windows XP Service Pack 2 with the same "ThinkPad | ThinkCentre" blue world map background. Both come with Microsoft Interactive Training, assorted ThinkVantage software, Symantec Norton anti-virus, and Diskeeper buggy scareware disk defragmenter.

(*) If you used the T61 discs but would like the IBM logo in the system properties, back up the files OEMINFO.INI and OEMLOGO.BMP from C:\Windows\System32, then replace them with the ones from the IBM folder in this zip file.

Of course, all included drivers are from 2005-2007, so it's a good idea to install updated drivers.
